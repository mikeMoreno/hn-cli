# hn-cli

A simple script for reading Hacker News on the command line.

```
Usage: hn-cli.py [OPTIONS]

Options:
  -p, --page INTEGER        Specify the page to display  [default: 1]
  -a, --article INTEGER     Specify the article to open
  -s, --submission INTEGER  Specify the submission to open
  --hide-ranks              Hide ranks on output
  --help                    Show this message and exit.
```
